
ファイルの追加は
　①ローカルにファイルを追加する(※ファイルはjis だとと文字化けする)
　②下記のコマンド
　　git add test.txt    untracked ⇒track　の状態にする
　　git commit -m "新しくファイルを追加した" コミットする

ローカルレポジトリからgithub(リモートレポジトリ)への反映は
　①下記のコマンド
　　git push

github(リモートレポジトリ)からローカルレポジトリへの反映は(ローカルレポジトリ作成後)
　①下記のコマンド
　　git pull

こまった(状態を確認したい時は)ときは
　①下記のコマンド
　git status
